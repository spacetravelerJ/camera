# camera

This codes are communication for RaspberryPi and window through google drive.
It is trying to detect a hand and recognize meaning of the gesture.
RaspberryPi send the image to Window Vscode.
In vscode, the KNN algorithm model is detect the hand and hand back the result to RaspberryPi.
In RaspberryPi, if there is correct commend in the result, then it will work that depends on the result.
